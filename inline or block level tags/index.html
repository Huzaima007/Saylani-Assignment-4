-------Block-Level Elements-------

  1. h1

    2. Creates the main heading (largest heading).

    Example:

    Main Heading

  2. p

    3. Creates a paragraph of text.

    Example:

    This is a paragraph.

  3. ol

    4. Creates an ordered list (numbered list).

    Example:
    1. First item
    2. Second item

  4. ul

    5. Creates an unordered list (bulleted list).

    Example:
    • First item
    • Second item

  5. hr

    6. Creates a horizontal line.

    Example:

  6. div

    7. Creates a container for other elements.

    Example:
    This is a container.

  7. table

    8. Creates a table.

    Example:
    Name Age
    Huzaima 16

  8. iframe

    9. iframe is used for embeds a video.

    Example:

  9. li

    10. Creates a list item. Used inside ol or ul.

    Example: This is a list item.

  10. pre

    11. Displays preformatted text (keeps spaces and line breaks).

    Example: 
          This text
          is preformatted.

-------Inline Elements-------

  1. br

    2. Starts a new line.

    Example:
    Hello
    World

  2. i

    3. Makes text italic.

    Example: This is italic

  3. b

    4. Makes text bold.

    Example: This is bold

  4. em

    5. Emphasizes text (usually makes it italic).

    Example: This is emphasized

  5. strong

    6. Strongly emphasizes text (usually makes it bold).

    Example: This is strongly emphasized

  6. sub

    7. Makes text smaller and lower than normal text. Used in chemical formulas.

    Example: H2O (for water)

  7. sup

    8. Makes text smaller and higher than normal text. Used in footnotes.

    Example: Note[1]

  8. img

    9. Shows an image.

    Example:

  9. button

    10. Creates a clickable button.

    Example:

  10. a

    11. Creates a link to another page.

    Example: Go to Example.com
